# 项目上下文文档

## 目录概述

本目录是**榆北选煤班组人员安全信息智能交互平台**的产品设计文档与原型开发项目目录。该目录包含完整的产品建设方案文档、各子系统的详细设计文档以及可交互的HTML原型页面，用于指导项目的开发、实施和管理。

## 项目类型

**产品设计文档与原型开发项目** - 包含产品设计文档、HTML/CSS/JS原型页面

## 项目背景

本项目是针对**榆北选煤分公司**及其下辖的四个选煤中心(小保当选煤中心、曹家滩选煤中心、涌鑫安山选煤中心、涌鑫沙梁选煤中心)设计的班组人员安全信息智能交互平台。

### 核心设计理念

- **"1+5+1"设计思路**:
  - 1个核心: 班组会议管理
  - 5大业务系统: 智能学习地图、AI知识库、职工安全档案、职工动态管理、安全积分
  - 1个落脚点: 安全生产责任制考核

- **三个转变**:
  - 从被动响应向主动预防转变
  - 从经验判断向数据驱动转变
  - 从统一管理向个性化互动转变

## 七大核心业务系统

### 1. 班组智能会议管理系统
- **班前会**: 智能签到、身心健康确认、风险辨识、安全宣誓
- **班后会**: 安全情况复盘、隐患整改反馈、问题分析
- **班组学习会**: 学习内容导入、案例研讨、规程解读

### 2. 班组智能学习地图系统
- 岗位能力建模与个性化学习路径
- 学习资源管理(岗位题库、双述实操库、应急演练库等)
- 学习效果评估与考核联动

### 3. 班组AI知识库管理系统
- 安全制度、工艺规程集中管理
- 智能检索与自动推送
- AI问答功能

### 4. 班组职工安全档案管理系统
- "一人一档"全生命周期管理
- 五类档案: 基本信息、安全行为、培训考核、健康状态、奖惩积分

### 5. 班组职工安全动态信息管理系统
- 信息采集-风险评估-关怀干预闭环
- 多渠道采集与隐私保护

### 6. 班组安全账户积分管理系统
- 工种差异化积分规则
- 积分公示与申诉机制
- 多维联动激励

### 7. 安全生产责任制考核管理系统
- 岗位定责-软件考核-结果应用闭环
- 差异化责任制清单
- 考核结果联动应用

## 页面开发规范

### 页面结构标准
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题 - 系统名称</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css" rel="stylesheet">
    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
</head>
```

### 导航栏标准
- 顶部固定导航栏 (sticky-top)
- 左侧: 系统名称和Logo
- 中间: 主要功能菜单（支持下拉菜单）
- 右侧: 用户信息、设置按钮

### 内容区域布局
- 使用 `max-width: 1400px` 居中容器
- 统计卡片: 4列网格布局
- 主内容区: 2fr + 1fr 两栏布局
- 卡片样式: 白色背景、圆角12px、边框1px

### 交互组件
- 按钮: `btn-primary` / `btn-outline-primary`
- 徽章: `badge-success` / `badge-warning` / `badge-danger` / `badge-info`
- 表格: Bootstrap标准表格样式，表头背景色
